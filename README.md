# 🎮 The Game Hub

Welcome to **The Game Hub**, a centralized arcade featuring a collection of original, tactical, and procedurally generated web games. 

Every game in this repository is built entirely from scratch using **Vanilla HTML, CSS, and JavaScript**—no heavy game engines, no external libraries, and zero dependencies. Just pure, lightweight code designed to run instantly in any modern web browser.

🌐 **[Play the Games Live Here](https://shubham-zope.github.io/game-hub)**

---

## 🕹️ Featured Games

### 🛡️ Void Sentinel
**Genre:** Tactical Grid Roguelike
You are humanity's last Sentinel. Survive infinite procedurally generated sectors, manage your Action Points (AP) carefully, and utilize cover. Defeat the Void Lords to draft permanent upgrades and build a devastating tactical engine.
* **Features:** Infinite procedural levels, line-of-sight mechanics, permanent upgrade drafting, mobile-responsive UI.

---

## 🛠️ Tech Stack
* **HTML5:** Semantic structure and inline SVG graphics.
* **CSS3:** Modern CSS Grid, Flexbox, UI animations, and fully responsive mobile-first design.
* **JavaScript (ES6+):** Procedural generation algorithms, pathfinding logic, state management, and HTML5 LocalStorage for saving high scores.
